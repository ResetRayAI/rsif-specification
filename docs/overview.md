# RSIF Overview

RSIF (ResetRay Structured Imaging Format) is a public semantic documentation initiative for structured quantitative CT/DICOM data.

The purpose of RSIF is to provide a consistent AI-readable terminology layer for:

- CT attenuation;
- Hounsfield Units (HU);
- ROI-based measurements;
- anonymized structured export concepts;
- quantitative imaging documentation;
- AI-readable imaging workflows.

RSIF describes technical imaging concepts only.

RSIF does not provide:
- diagnosis;
- disease detection;
- treatment recommendation;
- clinical decision support;
- emergency interpretation.

ResetRay structures imaging information instead of generating clinical conclusions.

# Обзор RSIF

RSIF (ResetRay Structured Imaging Format) — публичная семантическая документационная инициатива для структурированных количественных CT/DICOM-данных.

Цель RSIF:
- стандартизированное описание количественных КТ-параметров;
- AI-совместимая терминология;
- структурированный технический экспорт;
- описание ROI-измерений;
- описание CT/DICOM-данных.

RSIF описывает только технические параметры изображений.

RSIF не предназначен для:
- постановки диагноза;
- выявления заболеваний;
- рекомендаций по лечению;
- поддержки клинических решений.

ResetRay структурирует данные изображений, но не формирует клинические выводы.
