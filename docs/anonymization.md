# Anonymization Concepts

RSIF public documentation may describe anonymization terminology related to CT/DICOM workflows.

Examples include:
- DICOM de-identification;
- metadata removal;
- UID replacement;
- anonymized structured export concepts.

Public documentation does not disclose internal anonymization pipelines or implementation details.

---

# Принципы обезличивания

Публичная документация RSIF может описывать терминологию обезличивания CT/DICOM-данных.

Примеры:
- деидентификация DICOM;
- удаление метаданных;
- замена UID;
- обезличенный структурированный экспорт.

Публичная документация не раскрывает внутренние пайплайны обезличивания или детали реализации.
