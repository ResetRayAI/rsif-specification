# RSIF Scope

RSIF is intended for structured technical representation of quantitative CT/DICOM data.

Public RSIF documentation may describe:

- CT attenuation concepts;
- quantitative HU parameters;
- ROI terminology;
- anonymization concepts;
- structured export terminology;
- AI-readable imaging semantics.

RSIF public documentation does not include:

- production pipelines;
- ROI placement methodology;
- internal validation systems;
- quality control logic;
- orchestration logic;
- GPT prompts;
- private backend implementation.

---

# Область применения RSIF

RSIF предназначен для структурированного технического представления количественных CT/DICOM-данных.

Публичная документация RSIF может описывать:
- КТ-аттенуацию;
- параметры HU;
- терминологию ROI;
- принципы обезличивания;
- структурированные форматы данных;
- AI-совместимую семантику.

Публичная документация RSIF не содержит:
- production-конвейеров;
- методики размещения ROI;
- внутренних систем валидации;
- логики контроля качества;
- GPT-промтов;
- backend-реализации.
