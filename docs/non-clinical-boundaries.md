# Non-Clinical Boundaries

RSIF is not a medical device and is not intended for clinical decision making.

RSIF does not:
- diagnose disease;
- classify pathology;
- determine treatment;
- replace physician interpretation;
- provide emergency recommendations.

Public RSIF documentation is limited to technical imaging concepts.

---

# Неклинические границы

RSIF не является медицинским изделием и не предназначен для клинического принятия решений.

RSIF не:
- ставит диагноз;
- классифицирует заболевания;
- определяет лечение;
- заменяет интерпретацию врача;
- предоставляет экстренные рекомендации.

Публичная документация RSIF ограничена техническими понятиями обработки изображений.
